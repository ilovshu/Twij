Twij
====
Twitter API Proxy in Java

重要信息
----

- 完全支持 Twitter Rest API v1.1。
- 配合 Tweetbot API 插件使用的所有功能正常。
- 其它客户端暂时没有测试，如果无法使用请尝试 Twip。
- 目前只支持单个用户。//哪个好心人来帮忙填一下这个坑？;)
- 理论上支持 Streaming，不过因为容器的响应缓存限制我自己暂时没有真正成功。//哪个好心人来帮忙填一下这个坑？;)

使用方法
----

设置以下的环境变量后部署即可。

- ConsumerKey
- ConsumerSecret
- AccessToken
- AccessTokenSecret
